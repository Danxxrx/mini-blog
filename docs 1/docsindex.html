Mini-Blog

Projekt zaliczeniowy – aplikacja webowa

Opis projektu

Mini-Blog to prosta aplikacja webowa umożliwiająca publikowanie artykułów oraz dodawanie komentarzy i odpowiedzi do komentarzy (1 poziom zagnieżdżenia).

Funkcjonalności

  • Dodawanie artykułów
  • Lista wszystkich artykułów
  • Widok pojedynczego artykułu
  • Komentarze do artykułów
  • Odpowiedzi na komentarze (1 poziom)

Technologie

  • Backend: Node.js + Express
  • Baza danych: SQLite
  • Frontend: HTML, CSS, JavaScript
  • Hosting: Render

Linki

  • Aplikacja (Render): https://mini-blog-pkl7.onrender.com
